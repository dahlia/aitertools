Changelog
=========

Version 0.2.1
-------------

Released on June 4, 2022.

 -  Fixed an off-by-one bug that `take()` had returned a non-empty async
    iterator when `count <= 0`.

Released on May 24, 2022.

 -  Added `cycle()` function.
 -  Added `repeat()` function.

 -  Fixed an off-by-one bug that `take()` had returned a non-empty async
    iterator when `count <= 0`.
